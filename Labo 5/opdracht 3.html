Opdracht 3

style attributen?

Vives bevat een inline style attributen die een id bevat.
Miras bevat ook inline style attributen, die het layout van het element vraagt
Vrt bevat inline style attributen die de id, layout en type text aanvragen

<style> blokken?

Vives bevat geen stijlblokken, alleen inline attributen.
Miras bevat wel stijlblokken, die geen stijlregels bevatten, waar de layout opstaat.
Vrt bevat stijlblokken, met meerdere stijlregels.

<link> verwijzingen naar aparte CSS-bestanden?

Bij vives worden de links in een stylesheet gestopt.
Miras gebruiken ook links in een stylesheet, maar ook als een icon.
Vrt gebruikt maar één keer een link met een stylesheet, anders gebruikt het een icon of een preload of iets anders.
